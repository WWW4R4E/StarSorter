# StarSorter 隐私策略

最后更新：2026年6月

---

## 概述

StarSorter 是一款帮助用户管理 GitHub 星标仓库的工具。我们重视您的隐私，本隐私策略说明我们如何处理您的数据。

## 数据收集

### 身份验证信息

为了访问您的 GitHub 数据，您需要提供 GitHub Personal Access Token。此令牌仅存储在您的设备上，用于向 GitHub API 进行身份验证。

### 应用数据

应用会在本地缓存以下数据以提供更好的用户体验：

- 您的 GitHub 星标仓库列表
- 仓库分类信息
- 应用设置（主题、语言等）
- 通知数据

### 代理配置

如果您配置了自定义代理，代理 URL 会存储在本地设置中。

## 数据使用

收集的数据仅用于以下目的：

- 显示和管理您的 GitHub 星标仓库
- 提供仓库分类和搜索功能
- 同步 GitHub 通知
- 保存您的应用偏好设置

## 数据存储

所有数据均存储在您的设备本地，不会上传到任何第三方服务器。

## 数据共享

我们不会向任何第三方共享您的个人数据。您的 GitHub 令牌仅用于向 GitHub API 进行身份验证。

## 安全

我们采取合理的措施保护您的数据安全。您的 GitHub 令牌以加密方式存储在设备上。

## 第三方服务

应用使用 GitHub API 获取您的仓库数据。有关 GitHub 的隐私政策，请参阅 [GitHub 隐私声明](https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement)。

## 更改本隐私策略

我们可能会更新本隐私策略。更新后，我们会在应用内通知您。

## 联系方式

如果您对本隐私策略有任何疑问，请通过 GitHub 仓库 [StarSorter](https://github.com/WWW4R4E/StarSorter) 联系我们。
